# 🚀 Backend com Node.js

Repositório criado para armazenar todas as atividades da disciplina de Backend do curso de Sistemas para Internet.

## 📚 Conteúdo

- Introdução ao Node.js
- APIs com Express
- Rotas (GET, POST, PUT, DELETE, PATCH)
- Organização de projetos
- Testes com curl

## 📁 Projetos

### ✅ API de Tarefas
- Criar tarefa
- Listar tarefas
- Atualizar tarefa
- Deletar tarefa
- Marcar como concluída

### 🎬 API de Filmes
- Cadastro de filmes
- Filtros por status

### 💊 API de Medicamentos
- Controle de horários
- Organização de rotina de uso

## 🛠 Tecnologias utilizadas

- Node.js
- Express
- JavaScript

## ▶️ Como rodar os projetos

```bash
npm install
node app.js
